# 🚀 Aprendendo Desenvolvimento Web

Este repositório reúne meus estudos e práticas sobre **HTML, CSS e JavaScript**.  
O objetivo é consolidar conceitos fundamentais e criar projetos simples para fixar o aprendizado.

---

## 📘 Conteúdos estudados

- **HTML** → Estrutura de páginas, tags principais e semântica.  
- **CSS** → Estilização, cores, fontes, layouts responsivos.  
- **JavaScript** → Fundamentos da linguagem e lógica de programação.  

---

## 🧩 Tópicos de JavaScript

- Funções  
- Arrays  
- Estruturas condicionais (`if / else`)  
- Laços de repetição (`while`, `do...while`, `for`)  
- Operadores (aritméticos, lógicos e relacionais)  

---

## 🎯 Objetivo

Praticar e evoluir no desenvolvimento web, criando pequenos projetos que utilizem:
- Estrutura com **HTML**  
- Estilo com **CSS**  
- Dinâmica e lógica com **JavaScript**
